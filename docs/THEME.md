# Theme

UltraBuild uses a neon forge theme.

## Palette

| Token | Hex | Purpose |
| --- | --- | --- |
| Void | `#060712` | background |
| Forge Navy | `#0f172a` | panels/code |
| Arc Cyan | `#00e5ff` | primary glow |
| Violet Core | `#7c3aed` | secondary glow |
| Flux Pink | `#ff2bd6` | accent |
| Verify Green | `#a3e635` | success |
| Text | `#f8fafc` | foreground |

## Mascot

Flux is the UltraBuild forge-dragon: friendly, fast, portable, and tool-aware. Assets are SVG so they render crisply on GitHub and across operating systems.
